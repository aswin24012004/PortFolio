Java Application

A web application developed using Java and JSP for managing student information allows users to perform CRUD operations. Users can submit a form via JSP to store student details, which are processed by a Java Servlet and saved into a MySQL database. JSP pages retrieve and display student records by querying the database, while updates and deletions are handled similarly through Servlets, enabling users to modify or remove records. The application also provides search functionality, allowing users to filter and view specific student details efficiently.
